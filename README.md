# depth-camera-simulation-V-rep-pcl
An easy simulation of depth camera based on vrep and PCL library

### 介绍
最近在做深度相机环境感知和重建的工作。利用Vrep构建场景，获取深度信息，再利用PCL库重建点云。很有意思，具体工作无法展示，这是一个简单demo，enjoy it！

### 软件
- V-rep pro edu 3.6.2
- vs2019 + pcl + opencv

### 参考
